console_menu<br />
linux console menu tool for system maintain<br />

命令行菜单工具<br />
将系统维护中常用操作命令定义成菜单形式，方便日常操作<br />

使用方法：<br />
1. 在 mam.cfg 中定义维护操作菜单<br />
   定义方法参考帮助文件 [mam-help.cfg](https://github.com/snake4bird/console_menu/blob/master/mam-help.cfg)<br />
2. 设置 mam 为可执行脚本<br />
   chmod +x mam<br />
3. 执行菜单操作<br />
   ./mam<br />
